¿Qué es este proyecto?

Este es un sitio web modular construido con Eleventy y HolyGrail5 que tiene dos tipos de layouts distintos para diferentes secciones.

Estructura de layouts

El proyecto utiliza dos layouts principales:

  • Layout Docs: Con sidebar para documentación y guías
  • Layout Dutti: Sin sidebar, diseño limpio para páginas principales

Navegación

Utiliza el sidebar izquierdo para navegar entre secciones de documentación.
